# 🌐 Social Media API

A high-performance, asynchronous social media backend engine built with **FastAPI**. This project provides a robust foundation for social networking features including user engagement, content sharing, and secure authentication.

## 🚀 Key Features

* **Asynchronous Architecture:** Fully non-blocking I/O operations for high concurrency.
* **JWT Authentication:** Secure OAuth2 with Password hashing and Bearer tokens.
* **Automated API Docs:** Interactive documentation via Swagger UI and ReDoc.
* **Data Integrity:** Robust data validation and serialization using **Pydantic**.
* **Follow System:** Logic for user-to-user relationships and personalized feeds.
* **Scalable CRUD:** Optimized endpoints for Posts, Comments, and User Profiles.

## 🛠️ Tech Stack

* **Framework:** [FastAPI](https://fastapi.tiangolo.com/)
* **Server:** Uvicorn (ASGI)
* **Language:** Python 3.10+
* **ORM:** SQLAlchemy (Asynchronous)
* **Database:** PostgreSQL / SQLite
